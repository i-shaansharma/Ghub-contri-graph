# Ghub Automator

Automatically generates and maintains GitHub contribution activity for testing, demonstrations, visualizations, and contribution graph experiments.

## Overview

GitHub Contribution Automator is a Node.js-based utility that creates automated commits based on configurable schedules and activity patterns. The project can be used to understand how GitHub contribution graphs work, test GitHub workflows, or generate sample contribution data for educational and experimental purposes.

## Features

* Automated commit generation
* Customizable contribution frequency
* JSON-based configuration
* Lightweight and easy to configure
* GitHub Actions compatible
* Contribution graph experimentation
* Simple Node.js implementation

## Project Structure

```bash
.
├── data.json          # Contribution configuration
├── index.js           # Main automation script
├── package.json       # Dependencies and scripts
├── package-lock.json
└── README.md
```

## Installation

Clone the repository:

```bash
git clone https://github.com/i-shaansharma/Ghub-contri-graph.git
cd Ghub-contri-graph
```

Install dependencies:

```bash
npm install
```

## Usage

Run the automation script:

```bash
node index.js
```

Modify the configuration inside `data.json` to customize commit frequency and contribution behavior.
